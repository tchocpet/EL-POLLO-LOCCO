# 🐔 EL POLLO LOCO

**El Pollo Loco** is a browser-based 2D jump-and-run game built with pure JavaScript, HTML, and CSS.  
Dive into a colorful desert world, fight enemies, collect items, and face the legendary boss in an action-packed adventure.

---

## 🎮 Gameplay

In this game you control a character who must survive in a dangerous environment filled with crazy chickens and obstacles.

- Move, jump and explore the level
- Collect coins and bottles
- Throw bottles to defeat enemies
- Fight your way to the final boss
- Win or lose based on your strategy and skill

---

## ✨ Features

- Smooth side-scrolling gameplay
- Animated character and enemies
- Multiple enemy types including a boss
- Collectibles with status updates
- Sound effects and background music
- Pause, restart and game-over screens
- Mobile touch controls for smartphones & tablets
- Story mode before starting the game
